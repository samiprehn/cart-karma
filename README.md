# 🛒 Cart Karma

**Find ethical alternatives to Amazon** — independent, cooperative, and secondhand.

Cart Karma is a single-page tool that helps you replace Amazon purchases with better options: indie stores, worker co-ops, B Corps, secondhand marketplaces, and local businesses.

## Features

- **Search by product** — type what you'd normally buy on Amazon and get curated alternatives
- **Browse by category** — 15 categories including Books, Clothing, Electronics, Outdoor, Food, Streaming, and more
- **Store types** — every result is labeled: Indie, Co-op, B Corp, Secondhand, or Local Finder
- **Direct search links** — jump straight to your search query on each alternative site
- **Giftful callout** — built-in recommendation for replacing Amazon Wishlists

## Categories

Books · Clothing · Electronics · Music · Outdoor · Home · Tools · Beauty · Food · Toys · Art & Crafts · Pets · Office · Pharmacy · Streaming

## How It Works

1. Type a product (e.g. "hiking boots", "novel", "headphones")
2. Cart Karma detects the category and surfaces the best ethical alternatives
3. Click **Website** to visit the store or **Search there** to search directly

No backend, no tracking, no API keys — just a single `index.html` file.

## Running Locally

```bash
open index.html
```

Or serve with any static file server:

```bash
npx serve .
```

## Tech

Pure HTML, CSS, and vanilla JavaScript. Zero dependencies.
